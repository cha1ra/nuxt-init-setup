# nuxt-init-setup
※作成途中  
Nuxt.jsを初期設定した後、毎回実施する処理をまとめておく


## How to use

### 1. Install Nuxt.js

```
$ npx create-nuxt-app app

```

### Move to project directory & Clone this repository

```
$ cd app
$ git clone https://github.com/cha1ra/nuxt-init-setup.git
```

### exec nuxt-init.sh

```
$ cd nuxt-init-setup
$ bash script.sh
```
